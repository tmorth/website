My Query Portfolio

Showcase of some SQL queries I have developed for various tasks.

Query 1: Employee Data Analysis

This query fetches all employees with salaries above a certain threshold and groups them by department.


SELECT department, COUNT(*) AS employee_count
FROM employees
WHERE salary > 50000
GROUP BY department
ORDER BY employee_count DESC;

Query 2: Customer Purchase History

This query retrieves the top 10 customers based on the total value of their purchases.


SELECT customer_id, SUM(total_amount) AS total_spent
FROM purchases
GROUP BY customer_id
ORDER BY total_spent DESC
LIMIT 10;

Query 3: Sales Trend Analysis

This query generates monthly sales trends for the current year.


SELECT DATE_FORMAT(order_date, '%Y-%m') AS month, SUM(order_total) AS total_sales
FROM orders
WHERE YEAR(order_date) = YEAR(CURDATE())
GROUP BY month
ORDER BY month ASC;
